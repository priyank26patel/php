Features:
=========
a)HIOX Event Calendar help maintain event schedules.
b)Single Admin with login to manage events.
c)Admin can easily add, update, delete schedules or events.
d)Default sorting of events based on schedule date.
e)Events older by 24hrs than current date will be deleted automatically.
f)Can easily set any color to the calendar headers and rows.
g)Event Calendar column size can be dynamically changed using UI.


How To Install:
===============
Please take 5 minutes time and read installation instructions carefully and
completely! This will ensure a proper and easy installation.

   a)unzip HEC_1_2.zip, you will get a folder \HEC_1_2\HEC with installation
files.

File permissions:
a) Change the file permission to (read, write, execute) 777 for the following
files (Only for LINUX users)
HEC/colors.php,
HEC/size.php,
HEC/passwo.php,
HEC/values.php

Setting the password:
  a) Open the file about.php, from your web server.
  b) Click on the option “Password” to provide the Admin user name, password.

Embedding Event Calendar:
 a) Click on the option “Get Code” link/menu.
 b) Click on the button “Select All” to copy the code given in textbox.
 c) Paste the code in between the <head></head> tags of your page.

Updating Events:
 a) Click on “Manage Events” to update or delete an event.
 b) Event those are older than 24 hours are deleted automatically.

Changing Colors:
   a) Click on “Modify Colors” to change the event table colors

Changing the Calendar size:
    a) Click on “Cal Size” to change the size of the calendar.

Configurable look and feel:
 a) To change headersize, fontsize and date format, edit the file HEC_1_2\HEC\
calendar.php 
 b) Following are the variables in the first few lines to be changed.
$headersize = "15px";
$fontsize = "14px";
$dateformat = "m-d-y";
