README:
*******

Features:
---------
1) Decrements the count from the time limit to zero.

2) Refresh the page to start countdown from first.

3) Provides you the alert "Time Out", once after the set time is completed.

4) Options to set desired timer limit.

5) Responsive and easy to integrate.

Usage:
------
1) Download and unzip the file.

2) You will get index.php,jquery.js,script.js,Readme.txt and style.css file and images folder.

3) Run the index.php file in your browser.

Index.php:
----------
1.This file contains the entire functionality of "PHP Countdown Timer Code".

jquery.js:
-----------
jquery.js file contains the jquery library functions that supports the jquery animations. Do not change the code in this file. If any changes are done in this file, then it will affect the entire functionality of the script.

script.js:
---------
1. script.js file contains entire functionality code of "PHP Countdown Timer Code".

2. The script will work only when the script.js file is included.

How to embed the script in to your webpage?
-------------------------------------------
1) Copy the content inside the index.php file and paste the code inside of your webpage where you want to display the effect.

2) You can replace the css properties according to your needs.

3) This script will get to date from user and display number of days,hours,minutes and seconds remaining from the compared dates.

4) Make sure that the  script.js,style.css and jquery.js files are placed in appropriate folder. Include these files in your page.
